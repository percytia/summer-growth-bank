☁️ 云同步设置（GitHub）
使用 GitHub Gist 私有存储实现跨设备数据同步。天然无 CORS，完全免费，无需额外注册。
在 GitHub → Settings → Developer settings → Fine-grained tokens 中生成，需要 gist 权限
所有设备使用相同的家庭码，数据即可互通。每个家庭自动创建一个私有 Gist
📖 如何获取 GitHub Token：
1. 打开 github.com/settings/tokens
2. 点「Generate new token」（Fine-grained）
3. Token name 填：growth-bank-sync
4. Expiration 选：90 days 或更长
5. Repository access：Only select repositories → 不选任何仓库（或选 public）
6. Permissions → Account permissions → Gist：选 Read and write
7. 点 Generate token，复制生成的 token 填入上方

💡 为什么用 GitHub Gist？
• ✅ 天然无 CORS（github.io 和 api.github.com 同源）
• ✅ 完全免费，无限存储
• ✅ 数据存为私有 Gist，只有你能访问
• ✅ 无需注册额外服务，你已有 GitHub 账号
• ⚠️ Token 存在浏览器本地存储中，不会上传到任何服务器 • 无需代理/VPN，数据安全合规
