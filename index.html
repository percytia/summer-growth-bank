🏠 家庭同步设置（NAS 自托管）
数据存放在你自家的绿联 NAS，同源访问、无需公网、隐私最高。
绿联 NAS 上部署后的地址。局域网填 http://192.168.1.23:8765；外网用 UGREENlink 域名（https）。结尾不要带 /
在 GitHub → Settings → Developer settings → Fine-grained tokens 中生成，需要 gist 权限
家庭码已安全内置，点一下即填好。GitHub 禁止把 Token 写进公开网页，Token 仍需各设备粘贴一次（家人私发你的 github_pat_... 串）。
所有设备使用相同的家庭码，数据即可互通
✅ 打卡 / 修改后会自动上传云端，无需手动操作；此按钮仅用于「立刻强制同步一次」。
🏠 NAS 自托管（推荐，主）：
1. 把 nas-server 文件夹拷到绿联 NAS
2. Docker 里用 Compose 启动容器（详见部署手册）
3. 上面「NAS 服务地址」填 http://192.168.1.23:8765
4. 家庭码所有设备保持一致，点「保存并同步」
5. 外网访问：在 NAS 开启 UGREENlink，用它给的 https 域名替换地址即可
• ✅ 数据只存自家 NAS，隐私最高，无需公网 IP
• ✅ 同源访问，无 CORS / 混合内容问题

☁️ GitHub Gist（备用）：
1. 打开 github.com/settings/tokens → Generate（Fine-grained）
2. Permissions → Account → Gist 选 Read and write，生成后填入上方
• ✅ 外网随时可达，作 NAS 的容灾/异地备用
• ⚠️ Token 仅存在浏览器本地，不上传任何服务器
